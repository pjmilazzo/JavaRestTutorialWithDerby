Java Restful Service Tutorial adapted from 308tube on youtube

This is how to create a Java Restful Service.
Please go to http://youtube.com/308tube to see the original tutorials

Pre-Reqs

Make sure Derby and Tomcat are installed and setup. Test with DerbyWithTomcat project. If additional details are needed,
there is a ".mht" file in the doc folder of the DerbyWithTomcat project that will get you going quickly. If you want to review without Derby,
there is another project called JavaRestTutorial which has no db needed at all.
